如何在 Windows 设置 Nginx 为服务？

现状

当前 Nginx 版本为：1.14.0

从 Nginx 官网下载来的Nginx压缩包解压后直接运行 nginx.exe 没有注册为服务。

目标

将 Nginx 注册为 Windows 服务。

步骤

  1. 将下载来的 Nginx 压缩包解压至 X:\path\to\nginx-1.14.0 。
  2. 下载 nssm 解压至任意位置。 （建议直接解压至 X:\path\to\nginx-1.14.0 目录下）。
  3. 运行 cmd 并切换工作路径至 nssm.exe 目录（注意当前操作系统位数）。
  4. 在 cmd 输入 nssm.exe install "Nginx" 并回车执行。
  5. 稍等片刻会打开 nssm 对话框，在对话框里选择 nginx.exe ，点击 Install 按钮。
  6. 打开本地服务就可以搜索到 Nginx 服务了。
