throttle和debounce的区别

throttle限制函数随着时间的推移可被调用的最大次数。如：每100毫秒最多执行一次函数

假设正常情况下你要在10秒内调用此函数1000次。如果使用throttle限制此函数每100毫秒执行一次，那么它最多执行100次。

(10s * 1000) = 10000ms

10000ms / 100ms = 100 最大调用次数

debounce限制函数不会再被调用，直到一定的时间过去。如：只有当100毫秒内，这个函数没有被调用过，调用它

想象一下一个函数在3秒内不断被调用1000次（如，绑定scroll事件且快速不断的滚动鼠标滚轮）。如果你使用debounce限制它的调用，它只会在你停止蹂躏鼠标滚轮时才会调用。

示例

这两个函数在Underscore和Lodash都有提供。

throttle:
    
        $('body').on('scroll', _.throttle(function () {
            // do something
        }))
debounce:
    
        $('body').on('scroll', _.debounce(function () {
            // do something
        }))
